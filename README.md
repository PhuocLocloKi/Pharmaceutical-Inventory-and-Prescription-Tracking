# 📖 About

Medicine Management System is a console application developed in **C** for managing pharmacy inventory.

The system helps users efficiently manage medicines through a simple and user-friendly interface.

---

# ✨ Features

- 💊 Add new medicine
- 📋 Display medicine list
- 🔍 Search medicine
- ✏️ Update medicine information
- ❌ Delete medicine
- 💾 Save & Load data from file
- 📦 Inventory management

---

# 🛠 Technologies

- C Programming
- Structures
- File Handling
- Modular Programming
- Dynamic Memory Allocation

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/PhuocLocloKi/Medicine-Management-System.git
```

Go to project

```bash
cd Medicine-Management-System
```

Compile

```bash
gcc *.c -o medicine
```

Run

```bash
./medicine
```
